Libraries: OpenCV,Pillow,Scipy,Scikit-learn,matplotlib
Languages: python
Main function: convert(img)

Your can only have python as your input file. The input file should not open a contents of server nor should it try to open a window.

  • It must have a function named convert that takes an image as an input and returns a filtered image.
  • There can be other functions in the file.
  • The file could have necessary imports of libraries only as mentioned above.
  • The file should not have any unnecessary imports.
  • The file should not have any unnecessary functions.
  • It should not try deleting or modifying any files.
  • It should not try to connect to remote server or send any user data elsewhere.
  • Your script will be monitored over the runtime.
The img in the function that will be of type numpy.ndarray i.e. the file format you get when you open a image using opencv. You will have to return it in the same type. You can convert it to different formats as per convience. Machine learning models can be used to Filter and edit the image. Please use scipy for the same. Do mind in user privacy and security.
